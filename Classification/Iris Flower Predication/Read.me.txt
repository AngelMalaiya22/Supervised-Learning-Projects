# 🌸 Iris Flower Classification using Machine Learning

This project is based on the famous **Iris Flower Dataset** from the **Scikit-learn library**. The goal of this project is to classify iris flowers into different species using Machine Learning techniques.

The dataset was directly loaded from the `sklearn.datasets` module instead of using a CSV file.

---

## 📌 Project Overview

In this project, a Machine Learning model is trained to predict the species of an iris flower based on features like:

* Sepal Length
* Sepal Width
* Petal Length
* Petal Width

The model classifies flowers into:

* Iris Setosa
* Iris Versicolor
* Iris Virginica

## 📂 Dataset Information

The dataset used in this project is the built-in Iris dataset provided by Scikit-learn.

Dataset loaded using:

```python

```

This dataset contains:

* 150 samples
* 4 input features
* 3 flower classes
