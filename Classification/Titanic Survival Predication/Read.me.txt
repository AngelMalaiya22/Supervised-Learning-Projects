# 🚢 Titanic Survival Prediction using Machine Learning

This project uses Machine Learning techniques to predict whether a passenger survived the Titanic disaster or not based on different passenger details.

The project is implemented using Python and popular data science libraries in a Jupyter Notebook.

---

## 📌 Project Overview

The main objective of this project is to:

* Analyze Titanic passenger data
* Perform data preprocessing and visualization
* Train a Machine Learning classification model
* Predict passenger survival
* Evaluate model performance

This project is one of the most popular beginner Machine Learning projects and helps in understanding the complete ML workflow.

---

## 🛠️ Technologies Used

* Python
* Jupyter Notebook
* NumPy
* Pandas
* Matplotlib
* Seaborn
* Scikit-learn

---

## 📂 Dataset Information

The dataset contains passenger information such as:

* Passenger Class
* Name
* Gender
* Age
* Fare
* Embarked Location
* Number of Siblings/Spouses
* Number of Parents/Children

The target variable is:

```python
Survived
```

where:

* `0` = Did Not Survive
* `1` = Survived
