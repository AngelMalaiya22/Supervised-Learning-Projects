# Salary Prediction using Machine Learning

## Overview

This project focuses on predicting employee salaries using Machine Learning regression algorithms. The dataset contains information such as age, education level, job title, years of experience, and salary.

The main objective of this project is to:

* Perform data preprocessing
* Handle categorical data
* Train multiple regression models
* Compare model performances
* Visualize important patterns in the dataset

---

# Dataset Information

The dataset contains the following features:

| Feature             | Description                     |
| ------------------- | ------------------------------- |
| Age                 | Age of the employee             |
| Education Level     | Highest education qualification |
| Job Title           | Employee job role               |
| Years of Experience | Total years of work experience  |
| Salary              | Target variable to predict      |

The `Gender` column was removed during preprocessing because it was not important for prediction.

# Machine Learning Algorithms Used

The following regression algorithms were implemented and compared:

1. Linear Regression
2. Decision Tree Regressor
3. Random Forest Regressor

---

# Data Preprocessing

The following preprocessing steps were performed:

* Removed unnecessary columns
* Handled missing values using `dropna()`
* Converted categorical features into numerical form using One-Hot Encoding
* Split dataset into training and testing sets
* Applied feature scaling where required

---

# Model Evaluation Metrics

The following evaluation metrics were used:

* Mean Absolute Error (MAE)
* Mean Squared Error (MSE)
* R2 Score

# Conclusion

This project demonstrated the importance of selecting appropriate machine learning algorithms based on dataset characteristics.

Key findings:

* Linear Regression was not suitable for this dataset because salary relationships were non-linear.
* Decision Tree and Random Forest performed much better.
* Decision Tree achieved the best overall performance with the highest R2 Score.
* Proper preprocessing and feature encoding were essential for achieving accurate predictions.

Overall, tree-based algorithms proved to be more effective for salary prediction tasks involving categorical and complex data patterns.
