# 🎓 Student Performance Prediction using Linear Regression

This project uses the **Linear Regression Algorithm** to predict student performance based on different academic and personal factors. The project is implemented using **Python**, **Pandas**, **NumPy**, **Matplotlib**, and **Scikit-learn** in a Jupyter Notebook.

---

## 📌 Project Overview

The main objective of this project is to:

* Understand how Linear Regression works
* Perform data preprocessing and feature engineering
* Train a machine learning model on student data
* Predict students' final grades
* Evaluate model performance using different metrics

This project is beginner-friendly and helps in understanding the complete machine learning workflow.

---

* Python
* Jupyter Notebook
* NumPy
* Pandas
* Matplotlib
* Seaborn
* Scikit-learn

---

## 📂 Dataset Information

The dataset contains information related to students such as:

* School
* Study time
* Failures
* Family support
* Activities
* Internet access
* Health
* Absences
* Previous grades
* Final Grade (G3)

The target variable used in this project is:

```python
G3
```

which represents the **final grade of students**.

---

## ⚙️ Steps Performed in the Project

### 1️⃣ Importing Libraries

Imported all required Python libraries for:

* Data handling
* Visualization
* Model building
* Evaluation

---

### 2️⃣ Loading Dataset

The dataset is loaded using Pandas.

```python
pd.read_csv("student_data.csv")
```

---

### 3️⃣ Data Preprocessing

Performed:

* Removing unnecessary columns
* Handling categorical data
* One-hot encoding using `get_dummies()`
* Checking missing values

---

### 4️⃣ Feature Scaling

Used `StandardScaler()` to normalize the input features.

---

### 5️⃣ Splitting Dataset

The dataset was divided into:

* Training Data
* Testing Data

using `train_test_split()`.

---

### 6️⃣ Model Training

Trained the model using:

```python
LinearRegression()
```

---

### 7️⃣ Prediction

The trained model predicts student final grades based on input features.

---

### 8️⃣ Model Evaluation

The model performance was evaluated using:

* Mean Absolute Error (MAE)
* Mean Squared Error (MSE)
* R² Score
